# 4.6 Notifications Specification

## Notifikasi Spam WhatsApp

**Notifikasi spam WhatsApp** adalah sistem yang mendeteksi pengguna yang mengirim pesan terlalu cepat (spam) dan otomatis mengirim peringatan via WhatsApp. Default setting adalah 1 pesan dalam 6 detik, dengan cooldown 1 menit antar peringatan. Semua parameter dapat dikonfigurasi melalui web interface tanpa hardcode, disimpan di database dan dapat diubah secara dinamis melalui API endpoint.
