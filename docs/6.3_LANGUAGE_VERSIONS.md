## 6.3 Language versions

Bagian ini menyajikan pernyataan faktual mengenai versi bahasa dan runtime yang digunakan dalam kode sumber proyek. Semua pernyataan di bawah ini diambil langsung dari berkas konfigurasi dan dokumentasi yang ada di repositori.

1. Versi runtime
- File `package.json` pada masing-masing komponen tidak memuat batasan versi Node.js (`engines`), sehingga tidak ada versi Node.js yang dipaksakan oleh konfigurasi proyek. Dependensi yang digunakan sesuai untuk runtime Node.js modern.

2. Versi framework dan pustaka utama (diambil dari `package.json`)
- Backend (`avevapi/package.json`): express (^4.18.2), better-sqlite3 (^12.4.1), bcrypt (^5.1.1), axios (^1.11.0), jsonwebtoken (^9.0.2), oracledb (^6.9.0), mysql2 (^3.6.0), pg (^8.16.3).
- Frontend (`frontend/package.json`): next (14.2.0), react (18.2.0), react-dom (18.2.0); tailwindcss tercantum pada `devDependencies`.
- WhatsApp Bot (`wa/package.json`): whatsapp-web.js (^1.33.2), puppeteer (^24.25.0), axios (^1.7.2).

3. Bahasa pemrograman dan format modul
- Backend dan bot ditulis menggunakan JavaScript untuk Node.js. Beberapa bagian backend dikonfigurasi sebagai modul ES (`type: module`), sedangkan paket bot menggunakan format CommonJS (`type: commonjs`).
- Frontend dibangun dengan React/Next.js; dependensi pengembangan mencantumkan TypeScript sebagai devDependency.

4. Rencana perubahan versi
- Tidak ditemukan catatan atau dokumen dalam repositori yang menyatakan rencana upgrade versi bahasa atau framework. Tidak ada roadmap versi atau tiket upgrade yang tercantum dalam dokumentasi proyek.

Pernyataan penutup
