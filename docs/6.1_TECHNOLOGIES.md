## 6.1 Technologies

Dokumen ini mencatat teknologi yang saat ini digunakan di dalam repositori dan menjawab pertanyaan mengenai persyaratan teknologi untuk calon penyedia jasa. Bagian ini disusun agar dapat dilampirkan ke dokumen resmi.

1) Teknologi yang terdeteksi di kode (faktual)
- Backend: Node.js, Express. Terdapat middleware autentikasi, rate limiter, dan penggunaan paket keamanan seperti `helmet`.
- Database: SQLite (`better-sqlite3`) — file-based database.
- Frontend: Next.js (React) dengan Tailwind CSS untuk styling.
- WhatsApp Bot: `whatsapp-web.js` yang menjalankan browser headless (puppeteer-like) untuk berinteraksi dengan WhatsApp.
- Dependencies & utilitas: axios untuk HTTP, qrcode untuk QR code, beberapa utilitas untuk audit/log.

Referensi singkat (lokasi kode)
- `avevapi` (backend)
- `frontend` (Next.js + Tailwind)
- `wa/index.js` (bot)

Catatan penutup
- Bagian ini mencantumkan teknologi yang ditemukan di kode. Tidak ada rekomendasi atau saran perbaikan di bagian ini.
